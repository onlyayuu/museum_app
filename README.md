# Museum App

Aplikasi museum sederhana untuk menampilkan koleksi karya seni. Proyek ini dikembangkan sebagai bagian dari studi

### Fitur Utama
* Melihat daftar koleksi karya seni.
* Melihat detail dari setiap karya seni.
